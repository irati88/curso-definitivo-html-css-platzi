Tipos de medidas en CSS

  1. Absolutas: Son medidas estáticas / Parámetros fijos
    • Píxeles: El valor no cambia, independientemente del tamaño de la pantalla
  2. Relativas: Son variables
    • % - Varían en función del tamaño de la pantalla
    • em (element) - Relativo al tamaño de fuente de un elemento
    • rem (root element) - Relativo al tamaño de fuente del elemento raíz
    • max-width / max height
    • min-width / min height
    • vw (viewport width) - Porcentaje relativo a la anchura del viewport
    • vh (viewport height) - Porcentaje relativo a la altura del viewport
