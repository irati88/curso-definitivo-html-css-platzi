Medidas relativas de fuente "Rem"


Se refiere al tamaño de fuente del elemento raíz (html), suele ser 16px


Ajustamos el font-size de "html" a 62.5% y así 1rem = 10px


  • 1.6 rem = 16px
  • 2rem = 20px
  • 3 rem = 30px
