CSS: Tipos de Display

  • Display Block: Ocupa el 100% del espacio horizontal disponible. Cada elemento irá en una línea nueva. Se pueden aplicar margin/padding (en los 4 lados), width y height.
  • Inline: Solo ocupan el espacio del contenido. Si hay hueco, seguirá añadiendo más contenido seguido. Se pueden aplicar margin/padding solo en iz-der. No se pueden aplicar width ni height.
  • Inline-Block: Permite aplicar margin, padding, width, y height. Permite que otros elementos se posiciones en línea horizontalmente si hay espacio suficiente.
