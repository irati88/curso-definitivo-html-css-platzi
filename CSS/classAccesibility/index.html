Accesibilidad Web

HTML Semántico para Proyectos Inclusivos

Semántica:

  • Utilizar las etiquetas HTML (Header, Footer, Main Content, Sidebar...) en lugar de muchos "div"
  • Las etiquetas nos indican dónde estamos, por ejemplo, a los lectores de pantalla para que el usuario sepa en qué sección de la web está y pueda navegar en ella

Medidas relativas para accesibilidad en texto web

Texto:

  • Px = Medida absoluta, no se podrá aumentar en caso de problemas de vista. No es accesible
  • Rem = Medida relativa, se puede incrementar el tamaño después y escalar el texto

Atributos HTML para Mejorar la Accesibilidad Web

  • Labels: Activa el input del form al hacer click
  • Alt: Atributo que se agrega a la etiqueta img para que el software lea la descripción de la imagen
  • Titles: Atributo que se agrega a la etiqueta a o img. El texto se muestra al hacer hover sobre la imagen o hipervínculo
