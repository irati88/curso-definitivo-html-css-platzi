Orden de Declaración en CSS

  • Importancia

    • Primero carga los estilos del navegador

  • Después los estilos de nuestro .css
  • Por último, las declaraciones de nuestro .css que tengan un !important

Selectores & Especificidad (left el más importante - right el menos)

  • !important => 1,0,0,0,0
  • Inline styles (el del navegador)=> 0,1,0,0,0
  • #id => 0,0,1,0,0
  • .class => 0,0,0,1,0
  • tag (etiquetas HTML) => 0,0,0,0,1
